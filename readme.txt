Kwit Buy Button enables users of a blog to quickly buy ingredients online that are mentioned in cooking recipes

== Description ==
Kwit Buy Button automatically adds a button under ingredients lists and directs user to Kwit.pl website which allows to compare prices from biggest online shops and buy grocery products mentioned in a recipe.

= More information =
Please visit the plugin website at https://kwit.pl/api/kup_w_kwit

== Installation ==

Download zip file with Kwit Button plugin and upload it into you Wordpress plugin folder(wp-content/plugins/) using Wordpress dashboard. Then activate the plugin in your plugin section. Button will be added under lists with ingredients. If you don't want to have the button in particular list just include phrase "nokwit" as one of the list position. If you have you personal user key from Kwit.pl you can enter it in "Settings" -> "Przycisk Kwit" in your admin panel.
